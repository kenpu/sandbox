# clj-latex

A Clojure library designed to ... well, that part is up to you.

## Usage

    mvn install:install-file 
        -Dfile=./jlatexmath-1.0.3.jar 
        -DgroupId=jlatexmath 
        -DartifactId=latexmath 
        -Dversion=1.0.3 
        -Dpackaging=jar 
        -DgeneratePom=true 
        -DcreateChecksum=true 
        -DlocalRepositoryPath=local_mvn_repo
